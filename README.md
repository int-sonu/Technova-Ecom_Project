Technova - Static E-Commerce Frontend

Technova is a static E-Commerce frontend application built using React, TypeScript, and Tailwind CSS.  
This project focuses on UI design  and frontend validation without backend integration.

 Features

- Responsive Homepage
- Product Listing Page
- Cart Page
- Checkout Page
- Login & Signup (Frontend validation only)
- Modern UI with Tailwind CSS

 Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router DOM
